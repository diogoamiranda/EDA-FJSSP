\doxysubsection{Detailed Description}
Struct to store an operation. 

An operation contains an identifier (@id), a job identifier (@job\+Id), a machine identifier (@machine\+Id), a execution time (@execution\+Time), an operation state (@finished) and a pointer ($\ast$next).
