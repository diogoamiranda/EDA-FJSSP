\doxysubsection{Detailed Description}
Estrutura para armazenar uma operação. 

Uma operação contem um identificador (@id), uma máquina que executará a operação (@maquina\+\_\+id) e o tempo total em unidades (@tempo\+Total) para executar a operação com essa máquina.
