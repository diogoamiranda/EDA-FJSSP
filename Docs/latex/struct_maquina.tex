\doxysubsection{Detailed Description}
Estrutura para armazenar uma máquina. 

Uma maquina contem um identificador (@id), um boolean (@disponivel) que identifica se a maquina está ou não disponivel para ser usada e um apontador para a proxima maquina (next).

Todas as máquinas estão disponíveis no instante t = 0.
